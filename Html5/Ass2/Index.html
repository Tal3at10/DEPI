About Semantic HTML

Semantic HTML introduces meaning to web content through elements like <header>, <nav>, <main>, and <footer>.

These elements help both browsers and developers understand the structure and meaning of the content, improving accessibility and SEO.

Key Features

  • Improved document structure
  • Better accessibility
  • Enhanced SEO
  • Easier maintenance
  • Future-proof code

Semantic Elements

Common Semantic Tags

  • <article> - Independent, self-contained content
  • <aside> - Content indirectly related to main content
  • <details> - Additional details user can view/hide
  • <figure> - Self-contained content like illustrations
  • <mark> - Highlighted text
  • <time> - Machine-readable date/time

HTML Elements Comparison

Element Semantic Non-Semantic Purpose
Header <header> <div class="header"> Introductory content
Navigation <nav> <div class="nav"> Navigation links
Main Content <main> <div class="main"> Primary content
Footer <footer> <div class="footer"> Closing content
